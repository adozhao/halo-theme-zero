### 主题特性  
- 多终端适配
- 支持暗色模式
- 适配官方`链接管理`插件
- 持续优化更新

### 主题安装
1. 点击 [链接](https://www.halo.run/store/apps) 进入Halo官方应用市场
2. 搜索该主题`Zero`并点击 `下载` 按钮
3. 在弹出的对话框中，输入您的网站地址，点击`安装`即可
